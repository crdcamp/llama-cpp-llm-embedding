# Local LLM Embedding

This project is a continuation of the [M4-local-llm-research-assistant
](https://github.com/crdcamp/M4-local-llm-research-assistant) repository. Essentially, I'm creating a full pipeline where data relevant to the given prompt is pulled from the internet and embedded into a model to help ensure more accurate answers.

Local LLMs are kinda dumb, so I'm just trying to make them smarter.
