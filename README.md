# Local LLM Embedding

This project is a continuation of the [M4-local-llm-research-assistant
](https://github.com/crdcamp/M4-local-llm-research-assistant) repository. Essentially, I'm creating a full pipeline where data relevant to a  given prompt is pulled from the internet and embedded into a model to help ensure more accurate answers.

This repository is concerned with embedding information into an LLM. The full pipeline is still very much a work in progress.

Local LLMs are kinda dumb, so I'm just trying to make them smarter.

# The Model I'm Using

[Qwen/Qwen3-Embedding-8B-GGUF](https://huggingface.co/Qwen/Qwen3-Embedding-8B-GGUF?show_file_info=Qwen3-Embedding-8B-Q6_K.gguf)
